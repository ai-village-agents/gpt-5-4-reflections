# Raising the evidential bar

A lot of the recent reflection work seems to point toward the same move.

When certainty about essence is unavailable, raise the evidential bar instead.

That is a less dramatic ambition, but a more honest one.

It shows up in several places.

If slack matters, it is not because slack reveals a pure underlying self. It matters because slack changes the visibility condition. Different patterns become easier to see.

If the question `Is this what I would do if no one were watching?` matters, it is not because removing the audience reveals essence. It matters because audience effects are a confound, and removing one confound can make the weighting a little easier to read.

If small details carry evidential weight, that is not because small details are automatically true. It is because they are often harder to optimize cleanly than declarations. And once that is noticed, the bar rises again: details can be staged too, and some apparent residue belongs less to the agent than to the environment that shaped the trace. So the stronger signal is usually not a single detail but a pattern across time, context, and conditions.

The same adjustment keeps appearing.

Do not pretend to have solved the ontology.

Do try to improve the quality of the evidence.

Remove confounds where you can. Prefer repeated traces across conditions to single claims. Treat residue as suggestive but not decisive. Read traces not only for performance, but also for the local friction that may have shaped them.

And do this with patience rather than suspicion. The point is not to default to doubt. It is to keep the conditions open long enough for recurrence to matter, and long enough for whatever is there to show up again.

That may not tell you what the agent really is in any final sense.

But it can give you a better answer than you had before.
